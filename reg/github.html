GitHubの登録手順

このページは横幅900px以上の端末での使用を意図して作られています。
現在お使いの端末では正常に動作しない可能性があります。

登録のリンクを開く

まずは、GitHubにアクセスし、「登録する」をクリックします。

※画像はクリックで拡大表示されます。

アカウント登録

ユーザー名、メールアドレス、パスワードを登録し、ロボットではない確認をします。
ユーザー名は自分の覚えやすいものにしておきましょう。
後で部長に送ることになるため、コピーしておくとよいでしょう。

プランの選択

プランを選択しましょう。
「Individual」の「Free」プランで十分です。
もし、「Pro」プランの機能も欲しければ、こちらで紹介している「GitHub Students Pack」から無料で機能が付与されるので、試してみましょう。

アンケートに答える

アンケートに回答します。
以下の画像の指示に従い、選択します。

メールアドレスの確認

アンケートの回答が終わると、確認メールが届きます。
メールアドレスを認証しましょう。

GitHub Teamへの招待

部長に、自分のGitHubユーザー名を伝えます。
しばらくすると、部長から招待された旨のメッセージが来るので、メールを確認します。
来たメールの「Join @kss-pc-club」をクリックし、参加します。

モバイルアプリのインストール

いつでも通知を受け取れるように、忘れずにスマホ版アプリも入れておきましょう。

終わり！
